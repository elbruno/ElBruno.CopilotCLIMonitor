# Marketing graphics

This project includes starter assets and standards for social posts and feature promotion.

## Required deliverables

1. **Social card (landscape)**  
   Size: `1200x630`  
   Usage: X/Twitter, LinkedIn, blog embeds

2. **Feature card (square)**  
   Size: `1080x1080`  
   Usage: social posts, gallery previews

3. **Feature screenshots**  
   Source: app dashboard + tray menu + CLI integration examples

## Asset location

- `images/marketing/social-card.svg`
- `images/marketing/feature-card.svg`
- `images/screenshots/` (captured product screenshots)

## Visual guidelines

- Use official logo variants from `images/logo/`
- Keep high contrast text and avoid clutter
- Highlight 2-3 primary product capabilities per card
- Include GitHub repo URL on final exports
