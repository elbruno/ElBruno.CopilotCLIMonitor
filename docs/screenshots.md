# Product screenshots

This page tracks canonical screenshots used in README and docs.

## Target screenshots

1. System tray icon visible with context menu opened
2. Dashboard window with recent events
3. Example toast notification (task completed)
4. `copilotclimon doctor` successful output in terminal

## Capture standards

- Resolution: at least 1920x1080
- Theme: Windows 11 default light mode
- Language: English
- Sensitive data: hidden or anonymized
- File format: PNG

## File locations

Store screenshots in:

```text
images/screenshots/
```

Notification icon source asset:

```text
images/notification-icon.svg
```

Recommended naming:

- `tray-context-menu.png`
- `dashboard-recent-events.png`
- `toast-task-completed.png`
- `doctor-command-success.png`

## README usage

After adding screenshot files, reference them in `README.md` under the Features section.
